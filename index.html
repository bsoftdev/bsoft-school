Cursos na nossa Academia

Aqui está uma lista com os cursos que são administrados na nossa academia, click em um curso para ver a sua descrição.

Curso de HTML5

Html

É uma linguagem de marcação de hipertexto, as páginas web ou seja os Websites são construidos ou estruturados com essa tecnologia. Ela é o esqueleto de um Website. O exemplo perfeito é página que estás lendo esse artigo agora mesmo,Ela foi estrutura com o HTML.

Neste curso apredenderás sobre como estruturar um Website, isso do básico ao avançado,, essa linguagem é muito simples de se aprender

Curso de CSS3

css 3

CSS são folhas de estilo em cascata, o HTML em si só é responsável pela marcação dos conteúdos. E a beleza, como as cores, tipo de fontes, posisionamento dos elementos, etc...é feita pela tecnologia CSS3.

Durante a caminhada do curso de CSS3 você apredenderás técnicas para a estilização ou tornar os websites mais bonitos e atrativos.

Curso de Javascript

Javascript

É uma linguagem de programação usada para criar páginas interetivas ou seja é usada para implementar funcionalidade avançadas à um websites, assim como animações, validações dos campos de formulários, etc...

O conceito acima é relativo ao seu uso no client-side, mas também hoje o Javascript pode ser usado no Server-side. Mas neste curso você estudará o Javascript para client-side. Então, prepare-se!

Curso de PHP

PHP

É uma linguagem de programação de computadores interpretada, livre e muito utilizada para gerar conteúdo dinâmico na Web.Trata-se de uma linguagem extremamente modularizada, o que a torna ideal para instalação e uso em servidores web.

Neste curso para além estudadr a linguagem PHP também estudaremos sobre banco dados usando o SGBD MYSQL.

Curso de MySQL

mysql

O programa 'MySQL' é um sistema de gerenciamento de banco de dados relacionais baseado em comandos SQL (Structured Query Language - Linguagem Estruturada para Pesquisas) que vem ganhando grande popularidade, sendo atualmente um dos bancos de dados mais populares, com mais de 4 milhões de instalações.

Neste curso apredenderá a analisar e a criar banco de dados relacionais com o MYSQL.
